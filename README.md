# rustypen.github.io

This repository contains the GitHub Pages website for the Rusty Pen project.

## Website Overview

The website serves as a landing page for the Rusty Pen project, providing:
- Project description and tagline
- Link to the main GitHub repository
- Information about the iOS version

## Access

Visit the website at: [rustypen.github.io](https://rustypen.github.io)

## Project Information

### Rusty Pen

> The pen is rusty. The mind isn't.

A vintage-inspired cross-platform writing application that brings the ritual of traditional writing to the digital age.
